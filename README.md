# My_PIXIV_Download


本身是想试着找一下PIXIV的chrome插件，希望能够方便地将每日榜下载到本地。然而由于每日榜包含有前日的图片，因此全下载下来不仅浪费时间，而且占用不少空间，因而希望找到一个好的方法记录下前日的rank，进行筛选。在此记录下主要的参考资料：
